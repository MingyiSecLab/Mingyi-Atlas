# Shared Playwright CLI

## Use When
- Browser automation needs session state, screenshots, tracing, or form interaction.

## Workflow
1. Use persistent storage state for authenticated multi-step workflows when needed.
2. Capture traces or screenshots for UI-dependent evidence.
3. Prefer stable selectors and visible text for interactions.
4. Record generated artifacts in the active session directory.

## Outputs
- Browser automation guidance and artifact references.
